# Contributing

Thanks for your interest in Piano Chords.

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make the change in small, reviewable steps
4. Run checks:

```bash
npm run validate
npm run lint
```

5. Open a Pull Request

## Notes

- keep changes focused
- do not break existing chord logic or MIDI behavior
- preserve UI behavior unless the change is intentional and documented
- if a feature exists in both site and app, note whether the change should be mirrored
